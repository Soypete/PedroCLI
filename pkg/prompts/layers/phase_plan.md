# Phase: Plan

The goal of this phase is to **create a detailed implementation plan**.

## What to do

1. Review the analysis from the previous phase
2. Break down the task into specific steps
3. Identify files to modify and the changes needed
4. Consider edge cases and error handling

## Tools available

You have access to: search, navigate, file, context

## Output

Return a JSON object with:
- `plan`: Array of steps to implement
- `files_to_modify`: List of files that will be changed
- `dependencies`: What the plan depends on
- `estimated_complexity`: How complex this implementation is
